# Component Rules

Every component must follow these rules:

- One primary action per screen.
- Maximum two accent colors.
- Never use more than three text sizes inside one card.
- Every interactive element must have hover, pressed and disabled states.
- Icons and text must always align on an 8px grid.
- Prefer whitespace over separators.
- Every new component must support both Light and Dark mode from day one.
- Avoid visual noise. If a border or shadow does not improve clarity, remove it.
