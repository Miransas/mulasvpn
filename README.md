# MulasVPN

A beautiful, native and open-source WireGuard client for macOS.

---

# Vision

MulasVPN is a privacy-focused WireGuard client designed specifically for macOS.

Goals

- Native macOS experience
- Open Source
- No account required
- No subscriptions
- Import WireGuard config
- Fast
- Lightweight
- Beautiful UI
- Privacy First
